# Контекст экосистемы PROMOSTAFF (визитка + панель)

Зафиксировано, над какими репозиториями ведётся работа по продукту «визитка агентства» и админка:

| Репозиторий | Назначение |
|-------------|------------|
| **promostaff-agency-maxbot** (этот репозиторий) | Бот-визитка в мессенджере **MAX**: webhook, сценарии визитки, воронка `agency_max_funnel`, админка `/admin`. |
| **promostaff-agency-bot** | Бот-визитка в **Telegram**: те же сценарии/UX в параллели с MAX. |
| **promostaff-bot** | **Веб-панель** PROMOSTAFF PRO (Flask), общая БД/интеграции с PRO, не путать с визиткой агентства. |

Изменения по производительности MAX API (HTTP keep-alive, быстрый ответ `/webhook`) — в **`max_client.py`** и **`app.py`** этого репозитория.
